Håll 2m avstånd

För att minska risken att sprida eller ta emot corona var noga med att ta avstånd från andra människor

Undvik kollektivtrafik

Undvik att resa med buss, tåg och spårvagn då detta är en av de största spridningarna av corona. ta istället cykeln eller gå om du har möjlighet

Undvik folkmassor

Bjud inte in eller gå på fester, engagemang eller andra med stora folkmassor.

Stanna hemma vid sjukdom

Vid minsta lilla symptom stanna hemma för att minska risken för andra att bli smittad så de kan skydda sina nära och kära hemma. Jobba hemma så mycket det går

Tvätta händer ofta

Det enda som tar bort corona är att tvätta händerna med tvål, om det inte finns möjlighet använd istället handsprit

Hosta och nys i armveck

För att skydda andra var noga med att inte skvätta ner dem med bakterier som kan leda till sjukdom

Munskydd

Om du känner dig sjuk använd munskydd för att minska chansen att smitta andra eller när det inte går att hålla avstånd från andra.

Var rädd om varandra

Ta hänsyn om andra och va försiktig när det gäller dig själv

Om du vill vet mer om Corona eller vilka symptom man får kan du läsa vidare på 1177 eller söka fler svar hos Folkhälsomyndigheten
